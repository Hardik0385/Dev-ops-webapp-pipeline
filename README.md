# DevOps WebApp Pipeline

This project demonstrates a complete DevOps CI/CD pipeline including:
- **Version Control**: GitHub
- **CI**: GitHub Actions
- **ChatOps**: Slack Notifications
- **CD**: Netlify (Static Deployment)

## How to use
- Push changes to `main` branch.
- Watch the pipeline run in GitHub Actions.
- Receive notifications in Slack.
- View the updated site on Netlify.
